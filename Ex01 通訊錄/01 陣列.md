# 陣列

## 變數與資料類型回顧
- `let` 用於**可以被改變**的變數
- `const` 用於**不可被改變**的常數
```JavaScript
let age = 18;
age = 19; //沒問題

const birthday = '0101';
birthday = 19; //報錯, 常數的值不能被改變
```
何時用變數？何時用小寫常數？何用大寫常數？  

可以放進這些變數、常數的資料類型(資料型別)有哪些？  

- 數值 number：`100`、`3.14`  
- 字串 string：`'大家好'`、`"歡迎光臨"`  
- 布林值 boolean：`true`、`false`  

思考：如果有 10 位學生，要用 JavaScript 記下姓名，該怎麼寫？
``` JavaScript
let student1 = '小美';
let student2 = '阿明';
let student3 = '大華';
// ... 一直寫到 ...
let student10 = '小莉';

console.log(student1);
console.log(student2);
console.log(student3);
// ... 一直寫到 ...
console.log(student10);
```
問題：
1. 這樣寫程式碼會不會很長、很重複？
2. 如果想把全部學生的名字『一起』顯示出來，要寫10行 console.log？
3. 如果你要顯示的是全校學生呢？
這是一個災難

因此，一個解決方法，處理同一類別、多個資料的方法，就是**陣列**  
變數像是一個單獨的盒子，陣列像是一個資料收納盒  
收納盒有一個總名稱，比如叫 `students`

``` JavaScript
let student[1] = '小美';
let student[2] = '阿明';
let student[3] = '大華';
// ... 一直寫到 ...
let student[10] = '小莉';

// 用陣列顯示所有學生的方法
for(let name of students){
    console.log(name)
}
```
總結，陣列：用一個變數，存多筆資料結構相同的資料，解決難以維護的問題

## 知道了陣列的概念，接下來討論怎麼操作  

如何創造一個空的陣列？
```JavaScript
let myArray = [];
let myArray2 = Array([]);

```

陣列宣告，初始化時有值  
```JavaScript
let fruits = ['蘋果', '香蕉', '橘子'];
console.log(fruits)
```  
怎麼讀出來？  索引從 `0` 開始  
```
fruits 陣列：
[ '蘋果',  '香蕉',  '橘子' ]
   0        1        2
```
```JavaScript
let fruits = ['蘋果', '香蕉', '橘子'];
console.log(fruits)
console.log(fruits[0])
console.log(fruits[1])
console.log(fruits[2])
``` 

不只讀出來，還可以寫入  
```JavaScript
fruits[1] = '西瓜'
console.log(fruits)
``` 

**思考如果讀出一個不存在的格子，會怎麼樣？`fruits[100]`**

怎麼知道一個陣列裡面放了多少東西？  
```JavaScript
let fruits = ['蘋果', '香蕉', '橘子'];
console.log( fruits.length ); 

//永遠取得最後一個
let lastIndex = fruits.length - 1;
console.log( fruits[lastIndex] ); // 印出 '橘子'

console.log( fruits[fruits.length - 2] ); //這個印出什麼？
``` 

在 JavaScript 的**陣列**，可以存放資料型別不同的資料  

```JavaScript
let fruit = ['蘋果', '斤', 20.0];
console.log(fruit)
```
使用範例
如果在處理資料時，想像成資料庫的每筆記錄，同一筆記錄就會有很多資料型別
同一個欄位，資料型別應該相同  
```JavaScript
let fruits = [
    ['蘋果', '斤', 40.0],
    ['香蕉', '根', 15.0],
    ['橘子', '個', 20.0]
];
console.log(fruits)
console.log(fruits[0][2])
console.log(`${fruits[1][0]}, 1${fruits[1][1]} ${fruits[1][2]} 元`)

//參考，另一個種處理方式
let fruit_name = ['蘋果', '香蕉', '橘子'];
let fruit_unit = ['斤', '根', '個'];
let fruit_price = [40, 15, 20];
let index = 1;
console.log(`${fruit_name[index]}, 1${fruit_unit[index]} ${fruit_price[index]} 元`)
```

動態地增加或減少內容  
增加一個新的內容 `push()` 
```JavaScript
let fruits = ['蘋果', '香蕉', '橘子'];
fruits.push('葡萄'); 
console.log(fruits); 
console.log(fruits.length); 
```
一次增加多個內容  
```JavaScript
fruits.push('芒果', '草莓');
console.log(fruits); // 現在有 6 種水果
```

指定位置的增加或刪除  
`splice(index, deleteCount, ...items)`
```JavaScript
let fruits = ['蘋果', '香蕉', '橘子'];
fruits.splice(1, 0, '西瓜');    //在 1 的位置, 增加一個內容 `西瓜`
console.log(fruits); 
arr.splice(2, 1);   //把索引為 2 的內容刪掉
```

## 實作: 購物清單
第一步：建立一個空白的購物清單。
```JavaScript
let shoppingList  = [];
```
第二步：用 push 把你想買的東西加進去，至少加三樣。  
```JavaScript
shoppingList.push('牛奶');
shoppingList.push('雞蛋');
shoppingList.push('麵包');
console.log(shoppingList); // 檢查一下
```
第三步：假設你後來發現預算不夠，決定不買最後一樣東西了，請用 pop 把它移除。  
```JavaScript
let removedItem = shoppingList.pop();
console.log(`我不買 ${removedItem} 了。`);
console.log(`現在的清單是: ${shoppingList}`);
```
