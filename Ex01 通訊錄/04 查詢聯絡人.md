第四部分：實作與討論：設計查詢聯絡人功能 (10分鐘)
目標： 引導學生思考如何從聯絡人清單中找出特定的人，為下一堂課的迴圈與函式做準備。

時間	內容重點提示
0-5 mins	1. 提出問題與討論
• 提問：「如果想找『王小明』的電話，你會怎麼做？」
• 引導學生描述「手動」的過程：從第一個開始，檢查名字，如果不是就看下一個，直到找到為止。
5-10 mins	2. 介紹概念與預覽
• 介紹 for 迴圈的概念（不深入語法），說明電腦就是用迴圈來做重複檢查。
• 虛擬碼示範：

```JavaScript
for (陣列裡的每一個聯絡人) {
  如果 (當前聯絡人的名字 === '我要找的名字') {
    就把這個聯絡人的電話印出來
  }
}
```

• 總結：預告下節課將學習如何用程式實現這個邏輯。

總結與建議
您的課程規劃非常扎實。這個設計在您的框架下：

填滿了時間：透過加入必要的預備知識回顧和引導式討論。

由淺入深：從變數 -> 陣列 -> 物件 -> 兩者結合 -> 問題解決。

注重實作：理論後立刻搭配實作，加深理解。

承先啟後：最後的查詢功能為下一堂課埋下完美伏筆，激發學生的學習動機。
