## 物件 Object 的基礎  

新問題：如何描述一個"人"？  
用陣列的方法  
```javascript
let personArray = ['王小明', 25, '02-12345678', 'ming@example.com',0.5, 0.5, 150];
```
怎麼知道，什麼欄位是什麼訊息？
用索引很方便，但不知道是什麼含意。陣列的"數字索引"缺乏語意，這是它的重大限制。  

使用 key-value 鍵值對來解決這個問題  

## 建立物件
語法：用大括號 `{}` 來建立物件。  
創造一個人物物件
```javascript
let person = {
  name: '王小明', // 'name' 是 key, '王小明' 是 value
  age: 25,        // 鍵值對之間用逗號隔開
  phone: '02-12345678'
};
```
"key"：是屬性的唯一識別名稱，就像字典的"單字"或抽屜的"標籤"
"value"：是該屬性對應的實際資料內容，就像字典的"解釋"或抽屜裡的"物品"

## 取得內容的方式 
1. 點記號：物件.鍵名  
```javascript
console.log(person.name); // '王小明'
console.log(person.age);  // 25
```  

2. 括號記號：物件['鍵名']  
```javascript
console.log(person['phone']); // '02-12345678'
let keyToAccess = 'age';
console.log(person[keyToAccess]); // 25 - 這用點記號是做不到的
``` 
點記號較方便，括號記號較靈活。  

## 動態
物件是動態的。不需要一開始就定義好所有屬性，可以隨時新增或修改
新增  
```javascript
// 新增 email 屬性
person.email = 'ming@example.com';
// 修改 age 屬性
person.age = 26;
console.log(person);
```

刪除  
```javascript
delete person.phone;
console.log(person.phone); // undefined
```

## 練習
建立描述自已的物件，至少有三種屬性 `name`, `hobby`, `city`  
用 console.log 列印  

```JavaScript
let me = {}
me.name = 'Andy Huang';
me.hobby = 'Coding';
me.city = 'Taichung';
console.log(me.name)
console.log(me.hobby)
console.log(me.city)
```
## 巢狀物件  
資料跟資料之間關係不只一層
"人"有"聯絡方式"，"聯絡方式"有"電話, Email, 地址"

如何表達這種關係？使用用巢狀物件
物件的屬性裡面，放入另一個物件

```JavaScript
let person = {
  name: '王小明',
  age: 25,
  contact: { // contact 屬性的值，是另一個物件
    phone: '0912345678',
    email: 'ming@example.com',
    address: {
      city: '台中市',
      district: '西區'
    }
  }
};
```

如何存取深層的資料？一層一層用點記號往下找
```JavaScript
console.log(person.contact.phone); // '0912345678'
console.log(person.contact.address.city); // '台中市'
```
怎麼理解比較快，對資料說"的"

## 複製的概念
這裡有陷阱，要區分淺層複製和深層複制的概念

淺層複制
```JavaScript
let person1 = {
  name: '小明',
  contact: { phone: '123' }
};
let person2 = person1; // 這不是複製！這只是讓 person2 指向同一個物件
person2.name = '小華';
console.log(person1.name); // '小華' - 啊！怎麼連 person1 也改了？
console.log(person1);
console.log(person2);

```
深層複制
```JavaScript
let person1 = {
  name: '小明',
  contact: { phone: '123' }
};
let person2 = structuredClone(person1);
person2.name = '小華';
console.log(person1);
console.log(person2);
```

## 實作：建立個人資料庫  
建立一個包含3個朋友的資料庫  
第一步：建立一個空陣列 `let friends = [];`  
第二步：建立3個朋友物件，每個物件都必須包含：  
- name (字串)
- age (數字)
- hobbies (陣列，包含至少2個興趣)
- contact (巢狀物件，包含 phone 和 email)
```JavaScript
//範例
let friend1 = {
  name: '陳大同',
  age: 26,
  hobbies: ['爬山', '看電影', '煮咖啡'],
  contact: {
    phone: '0912345678',
    email: 'datong@example.com'
  }
};
```
第三步：用 push 將3個朋友物件加入 friends 陣列  
第四步：完成以下要求
- 印出第二個朋友的名字。
- 印出第一個朋友的第三個興趣。
- 印出第三個朋友的Email。
- 為第二個朋友新增一個興趣 '游泳'。

## Json  
寫物件的寫法，很像純文字的資料  
這個資料的表達方式就和 Json 一樣  
JSON 是一種輕量級的資料交換格式  
JSON 是前後端溝通、儲存資料的通用格式 
比較一下  
```JavaScript
// JavaScript 物件
let jsObject = {
  name: "小明",
  age: 25,
  isStudent: true
};

// JSON 格式
{
  "name": "小明",
  "age": 25,
  "isStudent": true
}
```
差別在於：JSON 的 key **"必須"** 用雙引號，而且不能有函式等複雜類型
