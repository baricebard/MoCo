第二部分：物件基礎 (20分鐘)
目標： 讓學生理解物件是 key-value 的集合，並能處理簡單的巢狀結構。

時間	內容重點提示
0-10 mins	1. 物件核心概念
• 為何需要物件：陣列只能靠索引，物件可以用有意義的「鍵」來管理資料。
• Key-Value 配對：{ key: value }
• 宣告與存取：
- 宣告：let person = { name: '小明', age: 25 };
- 存取：點記號 (person.name)、括號記號 (person['age'])。
  
10-20 mins	2. 巢狀物件與實作
• 巢狀物件：物件的 value 可以是另一個物件。
• 實例：
```javascript
let person = {
    name: '小明',
    contact: {
        phone: '0912345678', 
        email: 'ming@example.com'
    }
};
```
- 存取：person.contact.phone
• 簡單練習：建立一個「書本」物件，包含書名、作者、出版年。
