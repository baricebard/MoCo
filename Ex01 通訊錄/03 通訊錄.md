第三部分：實作：聯絡人清單 (物件陣列) (30分鐘)
目標： 綜合運用陣列與物件，建立一個可動態管理的聯絡人清單。

實作要求：建立「聯絡人」物件，包含姓名、電話、email

時間	內容重點提示
0-10 mins	1. 講解與示範
• 定義單一聯絡人物件結構。
• 實例：
```javascript 
// 單一聯絡人物件 
let contact1 = { 
    name: '王小明', 
    phone: '02-12345678', 
    email: 'wang@example.com' 
};
```

• 建立「聯絡人清單」陣列，並將物件放入。

10-25 mins	2. 學生實作時間
• 讓學生建立自己的聯絡人清單陣列 contacts。
• 要求至少手動建立 3 個聯絡人物件在陣列中。
• 引導：如何使用 push() 加入第四個聯絡人？
25-30 mins	3. 成果檢視與小挑戰
• 請學生在 Console 中印出整個清單、第二個聯絡人的姓名、第一個聯絡人的電話。

✅ 實作檢查重點：

語法正確性：物件的大括號 {}、鍵值對之間的逗號 ,。

結構一致性：每個聯絡人物件是否都包含 name, phone, email 這三個屬性。

陣列操作：學生是否成功將多個物件放入一個陣列中。

資料存取：學生是否能正確使用 contacts[i].name 這樣的語法來讀取特定聯絡人的特定資訊。
