# 實作：聯絡人清單 (物件陣列)

結合物件及陣列的功能，實作行動裝置聯絡人管理系統

## 目標
用一個陣列來管理多個聯絡人，每個聯絡人都是一個結構清晰的物件 
這個就是手機的通訊錄  

## 規畫  
- name (字串)：聯絡人姓名 - 核心識別資訊
- phone (字串)：電話號碼 - 核心聯絡資訊
- email (字串)：電子郵件 - 核心聯絡資訊
```javascript 
// 單一聯絡人物件 
let contact1 = { 
    name: '王小明', 
    phone: '02-12345678', 
    email: 'wang@example.com' 
};
```
清單的部份，使用陣列  
```javascript 
// 建立一個空陣列，準備放聯絡人
let contacts = [];

// 將聯絡人物件加入陣列
contacts.push(contact1);

// 更常見的做法是直接在建構時放入
let contacts = [
  {
    name: '王小明',
    phone: '02-12345678', 
    email: 'wang@example.com'
  },
  {
    name: '陳小美',
    phone: '0912345678',
    email: 'mei@example.com'
  }
];
```
這個 contacts 陣列就是我們的迷你資料庫，聯絡人清單   

## 實作重點
如何取用  
```javascript 
console.log(contacts[0].name); // '王小明'
```
混合了陣列的索引 `[0]` 和物件的點記號 `.name`  

功能擴充：程式先問使用者要不要增加一個使用者 `yes/no`  

```javascript 
let answer = prompt('是否要加入第四個聯絡人？ (y/n)');

if (answer === 'y' || answer === 'Y') {
  // 如果使用者輸入 y 或 Y，執行這裡的程式碼
  let newName = prompt('請輸入姓名：');
  let newPhone = prompt('請輸入電話：');
  let newEmail = prompt('請輸入Email：');
  
  // 建立新的聯絡人物件
  let newContact = {
    name: newName,
    phone: newPhone, 
    email: newEmail
  };
  
  // 將新聯絡人加入陣列
  contacts.push(newContact);
  alert('聯絡人已新增！');
  
} else if (answer === 'n' || answer === 'N') {
  // 如果使用者輸入 n 或 N，執行這裡的程式碼
  alert('好的，不新增聯絡人。');
  
} else {
  // 如果使用者輸入其他東西，執行這裡的程式碼
  alert('輸入錯誤，請重新整理頁面再試一次。');
}
```
為了讓資料更正確，加入判斷式  
檢查電話號碼是否不是空字串  
```javascript 
if (newPhone === '') {
  alert('錯誤：電話號碼不能為空！');
} else {
  contacts.push(newContact);
  alert('聯絡人已新增！');
}
```
檢查 Email 是否包含 `@` 符號
```javascript 
if (newEmail.includes('@')) {
  // 如果 newEmail 包含 '@'，條件成立
  contacts.push(newContact);
} else {
  alert('Email格式不正確！');
}
```
進階部份：把程式加入 while 廻圈，讓使用者可以一直新增，直到他們選擇退出為止

## 成果檢查
1. 在 Console 印出整個 contacts 陣列，檢查結構是否正確
    `console.log(contacts);`
2. 印出清單中"第二個"聯絡人的"姓名"
    `console.log(contacts[1].name);`
3. 印出"第一個"聯絡人的"電話"
    `console.log(contacts[0].phone);`
4. 印出"最後一個"聯絡人的 Email 
    `console.log(contacts[contacts.length - 1].email);`

## 實作檢查重點：
語法正確性：物件的大括號 {}、鍵值對之間的逗號
結構一致性：每個聯絡人物件是否都包含 name, phone, email 這三個屬性
陣列操作：學生是否成功將多個物件放入一個陣列中
資料存取：學生是否能正確使用 contacts[i].name 這樣的語法來讀取特定聯絡人的特定資訊
