## 實作
一個通訊錄如果只能把所有人印出來，就像一本沒有目錄的電話簿  
我要找到"王小明"的電話。在目前的 contacts 陣列裡，該怎麼做?  
請思考如何完成以上目標？

思考過程：  
1. 從聯絡人清單的"第一個"人開始看  
2. 檢查這個人的名字：是不是"王小明"  
3. 如果是，任務完成，把電話印出來  
4. 不是，就"移動到下一個人"，回到步驟2繼續檢查  
5. 如果檢查到最後一個人都沒有找到，就回報"找不到這個人"  

這個過程包含了幾個關鍵動作："從頭開始"、"逐個檢查"、"條件判斷"、"重複執行"  

## 這個就是演算法  
演算法就是解決一個問題的"明確步驟序列"。上面的方法就是一個"線性搜尋演算法"  
所以現在把我們解決問題的邏輯，用 JavaScript 寫出來  

## 迴圈  
電腦最擅長的做重複的事情。讓電腦重複執行一段指令的工具，叫做"迴圈"  
最常用來處理陣列的是 `for` 迴圈  
`for` 迴圈會對陣列裡的每個元素，都執行一次我們寫入的任務  

## `for` 廻圈  
基本語法  
```JavaScript
for (初始化計數器; 條件式; 計數器更新) {
  敘述區塊;
  [continue;]
  [break;]
}
```
for 迴圈包含三個關鍵部分
1. 初始化：設定計數器的起點 (let i = 0)。
2. 條件式：設定迴圈何時停止 (i < contacts.length)。
3. 更新：設定計數器如何變化 (i++)。
for 廻圈有三種型式，

所以，我們要完成的
```JavaScript
for (陣列裡的每一個聯絡人) {
  如果 (當前聯絡人的名字 === '我要找的名字') {
    就把這個聯絡人的電話印出來
  }
}
```
按照設計的寫完，我們列出三種寫法  
```JavaScript
let prompt = prompt("請輸入要查詢的名字","")
for (let i = 0; i <= contacts; i++) {
  if (contacts[i].name === prompt) {
    console.log(`${contacts[i].name}: ${contacts[i].phone}`)
  }
}
```
```JavaScript
let contacts[];
let name = prompt("請輸入要查詢的名字","")
for (let index in contacts) {
  if (contacts[index].name === name) {
    console.log(`${contacts[index].name}: ${contacts[index].phone}`)
  }
}
```  
```JavaScript
let contacts[];
let name = prompt("請輸入要查詢的名字","")
for (let friend of contacts) {
  if (friend.name === name) {
    console.log(`${friend.name}: ${friend.phone}`)
  }
}
```
